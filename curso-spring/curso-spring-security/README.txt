

		Este proyecto se enseña a usar spring security: el proyecto esta divido:
		
			+ Capa de servicio que se encarga de la parte de acceder a la base de datos embebida.
			
				H2
				
				Ibatis
				
				Spring 3.0.6
			
			+ Capa de web que es la encarga de la parte vista usando las siguientes tecnologias.
			
				JSF 2
				
				PrimeFaces
				
				Spring Security 3.0.6
				
		
		El proyecto tiene 2 perfiles:
			
			+ -Ptomcat que es el perfil por defecto
			
			+ -Pglassfih para deploy en glassfish 3
